### 1. Level 3 : Cobolt -> Goblin


코드가 매우 짧다. 
```
int main()
{
    char buffer[16];
    gets(buffer);
    printf("%s\n", buffer);
}
```
어셈블리도 마찬가지로 딱히 특이점은 없다.
